Välkommen till kursen ITE130

Kursen om Interaktiva användargränssnitt

Kursöversikt

Kursen innehåller grundläggande teoretisk och praktisk kunskap som när nödvändig för att planera för och göra prototyper av ett interaktivt system. Kursen behandlar och exemplifierar centrala typer av komponenter, användargränssnitt, interaktiva system, samt verktyg och metoder som används vid framtagning av interaktiva system. Kursen behandlar även kopplingen mellan ett interaktivt tekniskt system, tänkta användare och ett mer övergripande socio-tekniskt system.

Läromål

  • Redogöra för digitala systems uppbyggnad i förhållande till funktion, användare och kontext
  • Beskriva logik och relationer för funktionaliteten hos ett digitalt system
  • Utforma visuella användargränssnitt utifrån relevanta teorier inom interaktionsdesign
  • Tillämpa praktiska verktyg som används inom interaktionsdesign vid utformning av visuella användargränssnitt
  • Redogöra för hur ett digitalt system samspelar med mänskliga, tekniska och samhälleliga aspekter utifrån ett informationsdesignperspektiv

Faktaruta: Git

Git är ett distribuerat versionshanteringssystem som hjälper dig hålla koll på ändringar i kod och samarbeta med andra. Några grundkommandon:

  • git init – starta ett repo
  • git add . – lägg till ändringar
  • git commit -m "meddelande" – spara en version
  • git push / git pull – synka mot fjärrrepo
